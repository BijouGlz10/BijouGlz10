# ¡Hola! Soy Bijou

Soy QA Engineer con experiencia práctica en pruebas manuales, pruebas de API, validación de bases de datos y fundamentos de automatización. Mi experiencia previa en servicios financieros fortaleció mi capacidad para analizar procesos, detectar inconsistencias y trabajar con información crítica.

Disfruto entender cómo funcionan los sistemas, organizar información compleja e identificar problemas antes de que impacten la experiencia del usuario. Actualmente continúo desarrollando mis habilidades en Quality Assurance, automatización y análisis de datos.

---

## Objetivo profesional

Busco desarrollarme como QA Engineer participando en equipos que valoren la calidad del software, la mejora continua y la experiencia del usuario. Mi objetivo es seguir fortaleciendo mis conocimientos en testing, automatización y tecnologías utilizadas en entornos ágiles.

## Proyectos Destacados

### Urban Scooter | Proyecto Final QA End-to-End | TripleTen (2025)

Realicé pruebas end-to-end para validar funcionalidades web y móviles, verificando flujos críticos de usuario y requisitos de negocio.

Diseñé casos de prueba y listas de comprobación, ejecuté pruebas funcionales, documenté defectos y validé cambios en la base de datos mediante consultas SQL.

**Tecnologías utilizadas:** Jira, SQL, Android Studio, DevTools, Web Testing, Mobile Testing, Test Case Design, Bug Reporting

---

### Urban Routes | Proyecto de Automatización | TripleTen (2025)

Desarrollé pruebas automatizadas para validar funcionalidades clave de una aplicación web y apoyar actividades de regresión.

Implementé escenarios automatizados utilizando Python y Selenium para verificar comportamientos esperados de la aplicación.

**Tecnologías utilizadas:** Python, Selenium, Pytest, Automation Testing, Regression Testing

---

### Urban Grocers | Proyecto de Pruebas de API | TripleTen (2025)

Diseñé y ejecuté pruebas para endpoints REST, validando solicitudes, respuestas y reglas de negocio mediante Postman.

Verifiqué la consistencia de los datos entre las respuestas de la API y la base de datos utilizando consultas SQL.

**Tecnologías utilizadas:** Postman, REST API Testing, SQL, JSON, Database Validation
